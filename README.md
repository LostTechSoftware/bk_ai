### 🔔 About the project

Essa é a inteligência artificial da Foodzilla, apelidada de Betsy. Sua função é calcular preferências para os usuários e também realizar funções que auxiliam em diversas áreas e funcionalidades da empresa.

### 🛠️ Built With

- [Nodejs](https://nodejs.dev)
- [Express](https://expressjs.com)

<!-- GETTING STARTED -->

## 🚀 Getting Started

Para iniciar o projeto é necessário seguir os passos abaixo:

### 📋 Pre-requisites

É preciso ter heroku-cli e yarn

- yarn

  ```sh
  npm install --global yarn
  ```

- heroku-cli

  ```sh
  npm install -g heroku
  ```

### 🔧 Installation


1. Clone the repo
   ```sh
   git clone https://github.com/LostTechSoftware/bk_clientes
   ```
2. Install packages
   ```sh
   yarn
   ```
3. Clone environment keys
   ```JS
   heroku config -a foodzilla-staging-server -s >> .env
   ```

### ✔ Run

```bash
yarn staging
```
